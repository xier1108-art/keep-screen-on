# 화면안꺼지게

Windows에서 화면이 꺼지거나 절전 모드로 진입하는 것을 방지하는 경량 트레이 앱입니다.

## 기능

| 기능 | 설명 |
|------|------|
| 화면 유지 | Windows 공식 API(`SetThreadExecutionState`)로 화면 절전 차단 |
| 시스템 트레이 | 창 없이 트레이에서만 동작, 숨겨진 앱 목록에 위치 |
| 상태 아이콘 | 녹색(유지 중) / 회색(일시 중지) 아이콘으로 상태 표시 |
| 일시 중지 / 재활성 | 트레이 메뉴 클릭 또는 단축키로 토글 |
| 단축키 | `Ctrl + Shift + K` 로 즉시 토글 |
| 자동 종료 타이머 | 30분 / 1시간 / 2시간 / 4시간 후 자동 일시 중지 |
| 배터리 절약 모드 | 배터리 사용 중으로 전환되면 자동 일시 중지 |
| 설정 저장 | 타이머·배터리 설정 자동 저장 (`%APPDATA%\keepawake\`) |
| 완전 종료 | 전원 설정 원상 복구 후 종료 |
| 중복 실행 방지 | 이미 실행 중이면 두 번째 실행은 자동 무시 |

## 사용 방법

### EXE 바로 실행

`dist\화면안꺼지게.exe` 를 실행하면 트레이에 바로 올라갑니다. 창은 뜨지 않습니다.

### 직접 빌드

```bat
build.bat
```

`build.bat` 을 더블클릭하면 필요한 라이브러리를 자동 설치하고 `dist\화면안꺼지게.exe` 를 생성합니다.

> **필요 환경:** Python 3.10+, pip

## 의존 라이브러리

| 라이브러리 | 용도 |
|-----------|------|
| [pystray](https://github.com/moses-palmer/pystray) | 시스템 트레이 아이콘 |
| [Pillow](https://python-pillow.org/) | 트레이 아이콘 런타임 생성 |

나머지(`ctypes`, `threading`, `json` 등)는 Python 표준 라이브러리.

## 기술 상세

- **화면 유지:** `SetThreadExecutionState(ES_CONTINUOUS | ES_SYSTEM_REQUIRED | ES_DISPLAY_REQUIRED)` — 마우스 지글지글 방식이 아닌 Windows 공식 API
- **단축키:** `RegisterHotKey` WinAPI (외부 라이브러리 불필요)
- **배터리 감지:** `GetSystemPowerStatus` WinAPI
- **빌드:** PyInstaller 단일 파일 exe (`console=False`, `upx=True`)
